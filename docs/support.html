Marketing Privacy

Basic Help

  • Open the menu bar item to pick a companion, change style, resize, or reset the pet.
  • Open Memory Center to import local content, write notes, and review your timeline.
  • In the Mac App Store build, activity capture is off by default until you explicitly enable it.

Troubleshooting

  • If the pet is not reacting, check whether the app is muted and try a different style.
  • If activity capture is not recording, enable it in settings first.
  • If you want keyboard and mouse activity tracking, grant Accessibility permission in macOS System Settings.
  • If imports are missing, confirm the file was selected through the system file picker.
