BuddyClaw Privacy
Marketing Support

What BuddyClaw stores

  • Imported files you choose to add
  • Manual notes you write inside the app
  • Local memory events such as timeline entries, review milestones, or companion interactions
  • App settings and companion preferences

What BuddyClaw does not do by default

  • It does not require account creation.
  • It does not sync your data to a cloud backend.
  • It does not send imported content, notes, or local memory to a remote service in the current Mac App Store build.

Activity capture

In the Mac App Store build, activity capture is off by default.

If you explicitly enable activity capture, BuddyClaw may record frontmost-app switches, activity rhythm, and global keyboard or mouse activity after you separately grant Accessibility permission in macOS.

This information stays on your Mac as part of the local memory experience.

The App Store build uses sandboxed, user-selected file access for imports.

Your control

  • Turn activity capture off at any time
  • Remove imported content from your local library
  • Delete app data from your Mac by removing BuddyClaw data files and the app itself
